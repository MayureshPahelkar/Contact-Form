# Contact-Form

This project is a simple Contact Form Web Application developed using HTML, CSS, and PHP. The main purpose of this system is to allow users to send messages through a website by entering their name, email address, and message.

The user interface is designed using HTML and CSS, providing a clean and user-friendly form where users can easily enter their details. When the user submits the form, the data is processed using PHP on the server side.

The PHP script collects the submitted information and stores it in a separate .txt file on the server. Each submission is saved with the user's name, email, and message so that the website administrator can review the messages later.

This project demonstrates basic concepts of form handling, server-side scripting, and file handling in PHP. It is useful for beginners to understand how data submitted from a web form can be processed and stored without using a database.

Key Features:

💠 User-friendly contact form interface

💠 fields for Name, Email, and Message

💠 Form submission handled using PHP

💠 User data stored in a text (.txt) file

 💠 Simple and lightweight implementation without a database

Technologies Used:

💠 HTML

💠 CSS

💠 PHP

💠 File Handling (.txt storage)
